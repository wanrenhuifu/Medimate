硝苯地平控释片说明书

【药品名称】
通用名称：硝苯地平控释片
英文名称：Nifedipine Controlled-release Tablets
商品名称：拜新同、心痛定

【成分】
本品主要成分为硝苯地平，化学名称：2,6-二甲基-4-(2-硝基苯基)-1,4-二氢-3,5-吡啶二甲酸二甲酯。

【性状】
本品为圆形双凸的坚硬膜衣片（渗透泵型控释片），外壳不溶性，药物释放后以完整药壳形式排出体外。

【适应症】
1. 高血压。
2. 冠心病：慢性稳定型心绞痛（劳累性心绞痛）。
3. 变异型心绞痛（Prinzmetal's angina）。

【规格】
30mg/片；60mg/片

【用法用量】
口服。每日一次，每次30mg或60mg。必须整片吞服，不可分割、咀嚼或压碎。服药时间不受进食影响。剂量应个体化，根据病情严重程度和患者反应调整。最大剂量每日90mg。

【不良反应】
1. 血管扩张反应（常见于治疗初期）：头痛、面部潮红、踝部水肿（与剂量相关，非心衰所致）。
2. 心血管：心悸、心动过速。少数患者可能出现心绞痛加重（反射性交感神经激活）。
3. 胃肠道：恶心、食欲不振、便秘。牙龈增生（长期使用、口腔卫生差者多见）。
4. 神经系统：头晕、乏力、嗜睡。
5. 其他：皮疹、瘙痒。

【禁忌】
1. 对硝苯地平或任何二氢吡啶类钙通道阻滞剂过敏者禁用。
2. 心源性休克患者禁用。
3. 不稳定性心绞痛（变异型心绞痛除外）和急性心肌梗死（起病4周内）患者禁用。
4. 妊娠期妇女禁用（可能抑制宫缩、致胎儿缺氧）。
5. 严重主动脉瓣狭窄患者禁用。
6. 与利福平合用禁用（利福平大幅降低硝苯地平血药浓度，使其无效）。

【注意事项】
1. 严重低血压患者慎用（收缩压<90mmHg）。
2. 心力衰竭患者慎用，需在医生指导下使用。
3. 肝功能不全患者需减量（硝苯地平主要经肝代谢）。
4. 突然停药可能导致反跳现象（高血压危象或心绞痛加重），应在医生指导下逐步减量。
5. 药物空壳可能出现在粪便中，属正常现象。

【药物相互作用】
1. 与利福平合用：禁止联用。利福平通过强效诱导CYP3A4，使硝苯地平生物利用度降低90%以上。
2. 与CYP3A4抑制剂（酮康唑、伊曲康唑、克拉霉素、葡萄柚汁）合用：显著升高硝苯地平血药浓度，增加低血压风险。服药期间禁止食用葡萄柚或饮用葡萄柚汁。
3. 与其他降压药合用：降压作用叠加，可能导致血压过低。
4. 与β受体阻滞剂合用：可能增加心力衰竭风险，联合使用需谨慎。
5. 与地高辛合用：可能增加地高辛血药浓度。
6. 与华法林合用：个别病例报告硝苯地平影响华法林抗凝效果，联用期间应监测INR。

【药理作用】
硝苯地平为二氢吡啶类钙通道阻滞剂，通过选择性阻断血管平滑肌和心肌细胞膜上的L型钙通道，抑制钙离子内流，从而扩张外周动脉和冠状动脉。主要作用为降低外周血管阻力和血压，改善冠状动脉血流。

【贮藏】
遮光，密封，不超过25℃保存。
