布洛芬缓释胶囊说明书

【药品名称】
通用名称：布洛芬缓释胶囊
英文名称：Ibuprofen Sustained Release Capsules
商品名称：芬必得、美林、布洛芬

【成分】
本品主要成分为布洛芬，化学名称：2-(4-异丁基苯基)丙酸。

【性状】
本品为缓释胶囊，内容物为白色球形小丸。

【适应症】
用于缓解轻至中度疼痛如头痛、关节痛、偏头痛、牙痛、肌肉痛、神经痛、痛经。也用于普通感冒或流行性感冒引起的发热。

【规格】
0.3g/粒

【用法用量】
口服。成人，一次1粒，一日2次（早晚各一次）。用于发热时，一次1粒，必要时每8小时一次。24小时不超过4粒。饭后服用可减少胃肠刺激。本品为缓释制剂，必须整粒吞服，不得咀嚼或压碎。

【不良反应】
1. 胃肠道反应：恶心、呕吐、腹痛、腹泻、便秘、胃烧灼感或消化不良，是最常见的不良反应。少数患者可能出现胃溃疡、消化道出血或穿孔。
2. 神经系统：头痛、头晕、耳鸣、嗜睡、视力模糊。
3. 过敏反应：皮疹、荨麻疹、瘙痒。罕见剥脱性皮炎、多形性红斑、中毒性表皮坏死松解症。
4. 肝肾功能：转氨酶升高、肾功能损害。
5. 血液系统：粒细胞减少、血小板减少、贫血。
6. 其他：水肿、高血压。

【禁忌】
1. 对本品或其他非甾体抗炎药（NSAIDs）过敏者禁用。
2. 活动性消化性溃疡或消化道出血者禁用。
3. 严重肝肾功能不全者禁用。
4. 严重心力衰竭患者禁用。
5. 妊娠晚期（第三孕期）妇女禁用。
6. 冠状动脉搭桥手术（CABG）围手术期疼痛禁用。

【注意事项】
1. 本品为对症治疗药，用于止痛不得超过5天，用于解热不得超过3天，如症状不缓解，请咨询医师。
2. 支气管哮喘患者、有消化性溃疡史者、高血压或心脏病患者慎用。
3. 孕妇及哺乳期妇女慎用。
4. 60岁以上老年患者慎用，使用最低有效剂量。
5. 避免与其他非甾体抗炎药（包括选择性COX-2抑制剂）合并使用。
6. 服药期间不得饮酒或含有酒精的饮料。

【药物相互作用】
1. 与阿司匹林或其他水杨酸类药物同用时，不增强疗效而增加胃肠道不良反应及出血风险。禁止与阿司匹林联用。
2. 与抗凝药（如华法林）同用，可能增加出血风险。联用时需监测凝血功能。
3. 与降压药（ACEI/ARB、利尿剂）同用，可能降低降压效果，增加肾损伤风险。
4. 与甲氨蝶呤同用，可能增加甲氨蝶呤的毒性。
5. 与锂剂同用，可能增加锂的血药浓度。
6. 与地高辛同用，可能增加地高辛的血药浓度。

【药理作用】
本品为非甾体抗炎药，通过抑制环氧合酶（COX）从而减少前列腺素的合成，发挥解热、镇痛和抗炎作用。

【贮藏】
遮光，密封保存。
