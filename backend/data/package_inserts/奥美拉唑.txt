奥美拉唑肠溶胶囊说明书

【药品名称】
通用名称：奥美拉唑肠溶胶囊
英文名称：Omeprazole Enteric-coated Capsules
商品名称：洛赛克、奥克

【成分】
本品主要成分为奥美拉唑，化学名称：5-甲氧基-2-[[(4-甲氧基-3,5-二甲基-2-吡啶基)甲基]亚磺酰基]-1H-苯并咪唑。

【性状】
本品为肠溶胶囊，内容物为白色或类白色肠溶微丸。

【适应症】
1. 胃、十二指肠溃疡。
2. 反流性食管炎。
3. 卓-艾综合征（胃泌素瘤）。
4. 与抗生素联用根除幽门螺杆菌（Hp）。
5. 非甾体抗炎药（NSAIDs）相关性胃黏膜损伤的预防。
6. 上消化道出血（静脉给药）。

【规格】
10mg/粒；20mg/粒

【用法用量】
口服。必须在餐前1小时整粒吞服，不可咀嚼或压碎。
1. 消化性溃疡：每次20mg，每日1-2次，疗程4-8周。
2. 反流性食管炎：每次20-40mg，每日1-2次，疗程4-8周。
3. 幽门螺杆菌根除（三联疗法）：奥美拉唑20mg + 阿莫西林1g + 克拉霉素0.5g，均每日2次，疗程7-14天。
4. NSAIDs胃黏膜保护：每日20mg。

【不良反应】
通常耐受良好，不良反应多为轻度和一过性：
1. 胃肠道：恶心、腹痛、腹泻、便秘、腹胀。
2. 神经系统：头痛、头晕、嗜睡、失眠。
3. 皮肤：皮疹、瘙痒。
4. 长期使用风险（>1年）：
   - 高胃泌素血症，胃底腺息肉。
   - 维生素B12吸收减少（胃酸缺乏影响B12释放）。
   - 低镁血症（罕见但可严重，表现为肌肉痉挛、心律失常）。
   - 骨折风险轻度增加（髋部、腕部、脊柱）。
   - 肠道感染风险增加（艰难梭菌相关性腹泻）。

【禁忌】
1. 对本品或任何苯并咪唑类药物过敏者禁用。
2. 严重肝功能不全者慎用，需减量。

【注意事项】
1. 使用前应排除胃恶性肿瘤（PPI可能掩盖症状、延误诊断）。
2. 长期服用者应定期监测血镁水平。
3. 肝病患者应减量，每日不超过20mg。

【药物相互作用】
1. 与氯吡格雷合用：奥美拉唑可能降低氯吡格雷的抗血小板活性（均通过CYP2C19代谢），增加心血管事件风险。必要时可换用泮托拉唑（影响较小）。
2. 与华法林合用：可能增强华法林抗凝作用，需监测INR。
3. 与甲氨蝶呤合用：PPI可能减少甲氨蝶呤排泄，增加其毒性。
4. 与酮康唑、伊曲康唑合用：胃酸减少→抗真菌药吸收减少。
5. 与地高辛合用：胃酸减少→地高辛吸收轻度增加。
6. 与他克莫司合用：可能增加他克莫司血药浓度。

【药理作用】
奥美拉唑为质子泵抑制剂（PPI），特异性作用于胃壁细胞H+/K+-ATP酶（质子泵），不可逆地抑制胃酸分泌的最后步骤，显著抑制基础和刺激状态下的胃酸分泌。作用持续超过24小时，每日服药一次即可有效抑制胃酸。

【贮藏】
遮光，密封，在阴凉干燥处保存。
