阿莫西林胶囊说明书

【药品名称】
通用名称：阿莫西林胶囊
英文名称：Amoxicillin Capsules
商品名称：阿莫仙、再林

【成分】
本品主要成分为阿莫西林，化学名称：(2S,5R,6R)-3,3-二甲基-6-[(R)-(-)-2-氨基-2-(4-羟基苯基)乙酰氨基]-7-氧代-4-硫杂-1-氮杂双环[3.2.0]庚烷-2-甲酸三水合物。

【性状】
本品为胶囊剂，内容物为白色或类白色粉末。

【适应症】
适用于对阿莫西林敏感的革兰阳性菌和革兰阴性菌引起的感染：
1. 上呼吸道感染：咽炎、扁桃体炎、鼻窦炎、中耳炎。
2. 下呼吸道感染：急性支气管炎、肺炎。
3. 泌尿生殖道感染：尿道炎、膀胱炎、肾盂肾炎。
4. 皮肤及软组织感染：疖、痈、蜂窝组织炎。
5. 幽门螺杆菌感染（与克拉霉素和质子泵抑制剂联用）。
6. 急性单纯性淋病。

【规格】
0.25g/粒（按C16H19N3O5S计）；0.5g/粒

【用法用量】
口服。成人一次0.5g，每6-8小时一次。一日剂量不超过4g。
儿童一日剂量按体重20-40mg/kg，每8小时一次。
肾功能减退者需调整剂量。
本品宜饭后服用，可减轻胃肠道反应。

【不良反应】
1. 过敏反应：皮疹（最常见）、荨麻疹、瘙痒、药物热。罕见但严重：过敏性休克、Stevens-Johnson综合征、中毒性表皮坏死松解症。
2. 胃肠道：恶心、呕吐、腹泻、伪膜性肠炎。
3. 血液系统：嗜酸性粒细胞增多、白细胞减少、血小板减少。
4. 肝肾功能：一过性转氨酶升高、间质性肾炎。

【禁忌】
1. 对青霉素类药物过敏者禁用。
2. 传染性单核细胞增多症患者禁用（用药后斑丘疹发生率极高）。
3. 有青霉素过敏性休克史者禁用。
4. 对头孢菌素类药物过敏者慎用（交叉过敏风险约5-10%）。

【注意事项】
1. 用药前必须详细询问药物过敏史，必要时做青霉素皮试。
2. 青霉素皮试阳性反应者禁用本品。
3. 疗程较长者应检查肝肾功能和血常规。
4. 孕妇及哺乳期妇女慎用。

【药物相互作用】
1. 与丙磺舒合用：丙磺舒抑制阿莫西林肾小管分泌，使阿莫西林血药浓度升高且半衰期延长。
2. 与甲氨蝶呤合用：青霉素类药物可能减少甲氨蝶呤排泄，增加其毒性。
3. 与口服抗凝药（华法林）合用：可能增强抗凝作用，需监测INR。
4. 与别嘌呤醇合用：皮疹发生率增加。
5. 与抑菌性抗生素（四环素、红霉素、磺胺类）合用：理论上可能产生拮抗作用。

【药理作用】
阿莫西林为半合成广谱青霉素类抗生素。通过与细菌细胞膜上的青霉素结合蛋白（PBPs）结合，抑制细菌细胞壁的合成，使细菌迅速溶解死亡。对大多数致病的革兰阳性菌和革兰阴性菌均有强大的杀菌作用。

【贮藏】
遮光，密封，在干燥处保存。
